## 1/2 - Circular Charts

Regardless of the discussion on the usefulness of pie charts, there are a number of settings that can be specified and/or enabled, including:

- legend (on/offf)
- legend position
- legend title (on/off)
- data label style
- data label percentage decimal places

### Pie Chart

``` json
{
  "visualStyles": {
    "pieChart": {
      "*": {
        "legend": [
          {
            "show": true,
            "position": "Top",
            "showTitle": false
          }
        ],
        "labels": [
          {
            "show": true,
            "position": "outside",
            "labelStyle": "Category, data value, percent of total",
            "labelDisplayUnits": 0,
            "labelPrecision": 0,
            "percentageLabelPrecision": 0
          }
        ]
      }
    }
  }
}
```

### Donut Chart

The same settings are also available for donut charts.

``` json
{
  "visualStyles": {
    "donutChart": {
      "*": {
        "legend": [
          {
            "show": true,
            "position": "Top",
            "showTitle": false
          }
        ],
        "labels": [
          {
            "show": true,
            "position": "outside",
            "labelStyle": "Category, data value, percent of total",
            "labelDisplayUnits": 0,
            "labelPrecision": 0,
            "percentageLabelPrecision": 0
          }
        ]
      }
    }
  }
}
```
